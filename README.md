Last updated: 2025-12-07

2. Data Collected
The App itself does not collect personal data that directly identifies the user.
However, certain third-party services used by the App may process data as described below.

3. Permissions Used

FOREGROUND_SERVICE / FOREGROUND_SERVICE_MEDIA_PLAYBACK
Used to keep music playback running reliably in the foreground.

INTERNET
Needed to download audio files from Firebase Storage and to connect with Spotify for premium users.

READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE
Used only to access and play audio files stored locally on the user’s device.
The App does not upload, copy, or transmit these files.

SYSTEM_ALERT_WINDOW
Allows the App to display overlays when needed for playback controls.
Used strictly for UI functionality.

WAKE_LOCK
Prevents the device from sleeping during audio playback to ensure uninterrupted usage.

VIBRATE
Provides optional haptic feedback within the App.

The App does not use these permissions for anything other than their intended functionality.

4. Third-Party Services
Firebase (Google LLC)
The App uses Firebase Storage to download audio files.
Firebase may collect technical data such as device identifiers, IP address, crash logs, and usage data, as described in Google’s Privacy Policy.
More information: https://policies.google.com/privacy

Spotify
For premium users, the App connects to the user’s Spotify account to enable playback through Spotify’s SDK.
Authentication and data handling related to the user’s Spotify account are managed directly by Spotify.
The App does not access the user’s Spotify credentials or personal information beyond what Spotify’s API provides for playback authorization.
More information: https://www.spotify.com/legal/privacy-policy/

The App does not share data with any other third parties.

5. Children’s Privacy
The App is not specifically directed at children.
